手动推送步骤（推荐）
===================

第一步：在文件夹空白处右键 -> "Git Bash Here"
（如果没有这个选项，直接打开 CMD 然后 cd 到项目目录）

第二步：依次输入以下命令（每行回车）：

git remote add origin https://github.com/uinaqx/artprice-challenge.git
git branch -M main
git push -u origin main

第三步：输入凭据
当提示输入时：
- Username: uinaqx
- Password: [你的 Personal Access Token]

Token 获取方法：
1. 打开 https://github.com/settings/tokens
2. Generate new token -> Tokens (classic)
3. Note 填: ArtPrice Deploy
4. 勾选 "repo"
5. Generate token
6. 立即复制（只显示一次！）
7. 粘贴到 Password 位置（右键粘贴或 Ctrl+Shift+V）

===================
或者使用 GitHub Desktop（图形界面，不用输命令）
1. 下载 https://desktop.github.com
2. 登录你的 GitHub 账号
3. File -> Add local repository
4. 选择 Kimi_Agent_艺术价格竞猜 文件夹
5. 填写提交信息，点击 "Commit to main"
6. 点击 "Publish repository"
7. 选择 "GitHub.com" -> 仓库名 artprice-challenge -> Publish
